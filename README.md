# CRM-система
## Описание
Пет-проект приложение для мониторинга личных показателей: финансы, тренировки, питание. Предоставляет API для учета и анализа данных, а также интерфейс взаимодействия через ТГ-бота. Основная цель проекта — объединить различные аспекты повседневного учета и предоставить удобную аналитику и статистику.
