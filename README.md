# CRM-система
## Описание
Приложение для мониторинга ряда параметров для упрощения жизни: финансы, тренировки, питание (на данные момент в реализации финансы)
---
## Стэк
- FastAPI
- Asyncio
- PostgresSQL
- Alembic
- SQLAlchemy
- Postman
- Logging
- Aiogram

## Основной функциона приложения
- Добавление, изменение, удаление категорий трат
- Добавление, изменение, удаление счетов оплаты
- Добавление, изменение, удаление транзакций
- Добавление, изменение, удаление пользователей
- Использование телеграм-бота для управления приложением
